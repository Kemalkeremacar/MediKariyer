# MediKariyer Proje Dokümantasyonu

Bu klasör MediKariyer projesinin geliştirme dokümantasyonunu içerir.

## 📁 Klasör Yapısı

- `steering/` - Geliştirme rehberleri ve standartlar
- `docs/` - Teknik dokümantasyon
- `settings/` - Proje ayarları

## 🚀 Hızlı Başlangıç

1. `steering/` klasöründeki rehberleri okuyun
2. `docs/` klasöründeki teknik dokümantasyonu inceleyin
3. Geliştirme standartlarına uygun kod yazın

## 📞 İletişim

Sorularınız için geliştirme ekibiyle iletişime geçin.
