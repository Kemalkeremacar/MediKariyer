# MediKariyer Dokümantasyon Sistemi

Bu klasör, sistem tarafından oluşturulan dokümantasyonları içerir.

## Klasör Yapısı

- `architecture/` - Proje mimarisi dokümantasyonları
- `api/` - API dokümantasyonları  
- `flows/` - Kullanıcı akışları ve süreç dokümantasyonları
- `roles/` - Rol ve yetki dokümantasyonları
- `impact/` - Etki analizi raporları
- `assets/` - Diyagramlar, şemalar ve görseller
- `exports/` - Dışa aktarılan dokümantasyonlar

## Otomatik Oluşturma

Bu dokümantasyonlar sistem tarafından otomatik olarak oluşturulur ve güncellenir:

- Kod değişikliklerinde mimari analiz
- API endpoint'leri değiştiğinde API dokümantasyonu
- Rol değişikliklerinde yetki matrisi
- Akış değişikliklerinde süreç haritaları

## Manuel Güncelleme

Bazı dokümantasyonlar manuel olarak güncellenebilir:
- İş süreçleri açıklamaları
- Kullanıcı rehberleri
- Best practice'ler
- Troubleshooting kılavuzları

## Versiyon Kontrolü

Tüm dokümantasyonlar versiyon kontrolü altındadır ve değişiklik geçmişi tutulur.
