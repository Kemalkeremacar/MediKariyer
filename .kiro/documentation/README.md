# Dokümantasyon Sistemi

Bu klasör MediKariyer projesi için kapsamlı dokümantasyon sisteminin ana dizinidir.

## Klasör Yapısı

- `backend/` - Backend dokümantasyon sistemi
- `frontend/` - Frontend dokümantasyon sistemi  
- `shared/` - Ortak bileşenler ve utilities
- `config/` - Sistem konfigürasyonları
- `templates/` - Dokümantasyon şablonları
- `assets/` - Statik dosyalar (resimler, diagramlar)

## Kurulum

Detaylı kurulum talimatları için `docs/setup.md` dosyasına bakınız.

## Kullanım

Sistem kullanımı hakkında bilgi için `docs/usage.md` dosyasına bakınız.
