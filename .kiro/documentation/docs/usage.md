# Kullanım Kılavuzu

## Giriş

MediKariyer Dokümantasyon Sistemi, proje mimarisini, kullanıcı rollerini, ekran akışlarını ve değişikliklerin etkilerini merkezi olarak yönetir.

## Ana Özellikler

### 1. Proje Mimarisi Dokümantasyonu

#### Mimari Görüntüleme
- Sol menüden "Mimari" sekmesine gidin
- Backend, Frontend, Mobile katmanlarını görüntüleyin
- Bileşen ilişkilerini inceleyin

#### Bileşen Detayları
- Herhangi bir bileşene tıklayın
- Bağımlılıkları görüntüleyin
- Dokümantasyon detaylarını okuyun

### 2. Kullanıcı Rolleri Yönetimi

#### Rol Görüntüleme
- "Roller" sekmesine gidin
- Admin, Doktor, Hastane rollerini görüntüleyin
- Yetki matrisini inceleyin

#### Yetki Düzenleme (Admin)
- Rol seçin
- "Düzenle" butonuna tıklayın
- Yetkileri ekleyin/çıkarın
- Değişiklikleri kaydedin

### 3. Ekran Akışları

#### Akış Görüntüleme
- "Akışlar" sekmesine gidin
- Rol bazlı akışları seçin
- İnteraktif akış diyagramını inceleyin

#### Akış Düzenleme
- Akış seçin
- "Düzenle" moduna geçin
- Adımları ekleyin/çıkarın
- Geçiş koşullarını tanımlayın

### 4. Etki Analizi

#### Değişiklik Analizi
- "Etki Analizi" sekmesine gidin
- "Yeni Analiz" butonuna tıklayın
- Değişiklik detaylarını girin:
  - Değişiklik tipi (UI, API, Database, vb.)
  - Etkilenen bileşen
  - Değişiklik kapsamı
- "Analiz Et" butonuna tıklayın

#### Sonuçları İnceleme
- Etkilenen bileşenler listesi
- Risk seviyesi değerlendirmesi
- Öneriler ve aksiyonlar
- Detaylı etki raporu

### 5. API Dokümantasyonu

#### API Explorer
- "API Docs" sekmesine gidin
- Endpoint'leri kategorilere göre görüntüleyin
- İnteraktif API test aracını kullanın

#### Endpoint Detayları
- Endpoint seçin
- Request/Response şemalarını görüntüleyin
- Örnek istekleri çalıştırın
- Hata kodlarını inceleyin

## Kullanıcı Rolleri ve Yetkiler

### Admin Kullanıcıları
- Tüm özelliklere erişim
- Kullanıcı yönetimi
- Sistem konfigürasyonu
- Rol ve yetki düzenleme

### Geliştirici Kullanıcıları
- Mimari dokümantasyonu görüntüleme
- Etki analizi yapma
- API dokümantasyonu görüntüleme
- Kod standartları erişimi

### Proje Yöneticileri
- Akış dokümantasyonu görüntüleme
- Etki analizi raporları
- İstatistik ve metrikler
- Proje durumu takibi

## Bildirimler

### Gerçek Zamanlı Bildirimler
- Dokümantasyon güncellemeleri
- Etki analizi sonuçları
- Sistem uyarıları
- Kullanıcı aktiviteleri

### E-posta Bildirimleri
- Günlük özet raporları
- Kritik değişiklik uyarıları
- Sistem bakım bildirimleri
- Haftalık aktivite raporu

### Bildirim Ayarları
- Profil → Bildirim Ayarları
- Bildirim tiplerini açın/kapatın
- E-posta sıklığını ayarlayın
- Özel filtreler oluşturun

## Arama ve Filtreleme

### Genel Arama
- Üst menüdeki arama kutusunu kullanın
- Tüm dokümantasyonda arama yapın
- Sonuçları kategorilere göre filtreleyin

### Gelişmiş Filtreleme
- Her sayfada filtreleme seçenekleri
- Tarih aralığı filtreleme
- Yazar bazlı filtreleme
- Kategori ve etiket filtreleme

## Dışa Aktarma

### PDF Raporu
- Herhangi bir sayfada "Dışa Aktar" butonuna tıklayın
- PDF formatında rapor oluşturun
- Özel şablon seçin

### Excel Verileri
- Tablo verilerini Excel formatında indirin
- Filtrelenmiş verileri dışa aktarın
- Özel sütun seçimi yapın

## Kısayollar

### Klavye Kısayolları
- `Ctrl + K`: Hızlı arama
- `Ctrl + N`: Yeni dokümantasyon
- `Ctrl + S`: Kaydet
- `Ctrl + Z`: Geri al
- `Esc`: Modal kapat

### Hızlı Erişim
- Son görüntülenen sayfalar
- Sık kullanılan özellikler
- Kişisel yer imleri
- Özel dashboard'lar
